# CLAUDE.md

## Project Overview

Cartwheel Scorecard Architecture — a single-page HTML application that visualizes a scorecard framework with interactive filtering and layered criteria display.

## Tech Stack

- **Frontend**: Vanilla HTML/CSS/JavaScript (no build tools or frameworks)
- **Fonts**: Google Fonts (Montserrat, Inter)
- **Deployment**: Static file (`index.html`)

## Repository Structure

```
index.html   — Full application (markup, styles, and logic in one file)
README.md    — Project readme
CLAUDE.md    — This file
```

## Development Notes

- The entire app lives in a single `index.html` file with embedded `<style>` and `<script>` blocks.
- No package manager, bundler, or build step is required. Open `index.html` in a browser to run.
- When editing, preserve the existing code style: CSS at the top in a `<style>` block, JS at the bottom in a `<script>` block.
